Introduction

What's Web Audio API?

The Web Audio API is a high-level JavaScript Application Programming Interface (API) that can be used for processing and synthesizing audio in web applications.
The audio processing is actually handled by Assembly/C/C++ code within the browser, but the API allows us to control it with JavaScript.

What's Oscilloscope 1.0?

Oscilloscope 1.0 is a web application that will allow you to generate sounds with the Web Audio API in your browser.
This is based on Web Audio API Oscillators which will attempt to cover what oscillators are, how to create them, changing waveforms, setting frequency, and layering multiple oscillators together to create rich.
An oscillator is a circuit that produces a repetitive, oscillating signal.
Along with audio filters and audio controllers, audio oscillators are one of the basic elements required for modern analog subtractive synthesizers.

  • Type: is the shape and form of the signal that we are going to create.
  • Frequency: is the number of occurrences of a repeating event per unit time. (Hz).
  • Detune: it defines the standard frequency tuning. (Hz).
  • Start after: it indicates how many seconds later the sound must play

Keyboard Shortcuts

Shortcut Key for Start & Stop

  • Press [SPACE] to start/stop the oscilloscope.

Shortcut Keys for Type are S, Q, W and T

  • Press [S] to set the type to "Sine" (default)
  • Press [Q] to set the type to "Square"
  • Press [W] to set the type to "Sawtooth"
  • Press [T] to set the type to "Triangle"

Shortcut Keys for Frequency are 1, 2, and 3

  • Press [1] to set the frequency to 200 (default)
  • Press [2] to set the frequency to 500
  • Press [3] to set the frequency to 800

Shortcut Keys for Detune are 6, 7, and 8

  • Press [4] to set the detune to -30
  • Press [5] to set the detune to 0 (default)
  • Press [6] to set the detune to 50
